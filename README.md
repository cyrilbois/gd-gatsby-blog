# gd-gatsby-blog

My personal website.

## Installation

```sh
yarn
```

## Development

```shell
yarn dev
```

## Deploy on [Netlify](https://app.netlify.com)

```shell
yarn deploy
```

## Notes

When embedding tweets with [gatsby-plugin-twitter](https://www.gatsbyjs.org/packages/gatsby-plugin-twitter/?=), include only the `<blockquote>`, not the `<script>`.
